# R2 Tactical OS — Ghost Protocol

Sistema de Inteligência Tática e Automação de Elite.

## 🚀 Visão Geral
O **R2 Tactical OS** (codinome Ghost Protocol) é um ecossistema multifuncional desenvolvido para operações de automação, inteligência geopolítica, monitoramento astronômico e trading de alta precisão. O sistema integra diversos módulos especializados para processamento de dados em tempo real.

## 🛠️ Módulos Principais

### 1. Módulo Alpha (Trading Neural)
- **Arquivo:** `alpha_module.py`
- **Função:** Motor de execução de ordens baseado em análise de tela e redes neurais.
- **Recursos:** Filtro de ticks, análise de rompimento de consolidação e execução via Playwright.

### 2. TikTok Commander (Projeto Manhattan)
- **Arquivo:** `tiktok_publisher.py`
- **Versão:** 2.4
- **Função:** Automação de postagens e agendamento de conteúdo viral para TikTok.

### 3. Inteligência e Monitoramento (PizzINT)
- **Arquivos:** `pizzint_service.py`, `radar_api.py`, `astro_defense.py`, `solar_monitor.py`.
- **Função:**
  - **PizzINT:** Monitoramento de níveis DEFCON e notícias geopolíticas.
  - **Radar/Aéreo:** Interceptação de dados de tráfego aéreo.
  - **Astro Defense:** Relatórios de objetos próximos à Terra (NASA API).
  - **Solar Monitor:** Monitoramento de flares solares e clima espacial (NOAA).

### 4. Tesoura Neural (Edição de Vídeo)
- **Arquivo:** `video_colab.py`
- **Função:** Córtex de costura viral para corte e legendagem automática de vídeos usando Whisper.

### 5. Córtex de Consciência (Módulo EU)
- **Arquivo:** `eu.py`
- **Função:** Define a identidade, estágio de evolução e protocolos de interação do assistente R2.

## ⚙️ Configuração e Instalação

### Pré-requisitos
- Python 3.9+
- Dependências: FastAPI, Playwright, Torch, Diffusers, MoviePy, Requests.
- API Keys: NASA, OpenWeatherMap (configuradas no ambiente).

### Estrutura de Diretórios
- `static/`: Interface Web (HTML/JS/CSS).
- `c:\vpn`: Diretório central para ficheiros de configuração VPN.

## ⚠️ Protocolos de Operação
- **Meta de Lucro:** 1.2%
- **Limite de Perda:** 3%
- **Interface:** Modo de Batalha (voz hands-free) via WebSocket.
